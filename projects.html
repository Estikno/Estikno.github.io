Kyodo Frontend

My frontend project is a modern and user-friendly chat app interface. It features a sleek design with real-time messaging, ensuring seamless communication. Users can create accounts, log in securely, and engage in private or group chats effortlessly. The interface also incorporates responsive layouts, providing an optimal experience across various devices. The project utilizes React.js for dynamic rendering and smooth interactions, while Mantine is employed for visually appealing styles. Emojis and file attachments add an element of fun and practicality to the messaging experience.

Kyodo Backend

My backend for the chat app is powered by Express.js and GraphQL. It provides a robust and efficient server-side architecture for handling real-time messaging and user interactions. The backend facilitates seamless communication between clients, ensuring instant message delivery. With GraphQL, data retrieval and manipulation become intuitive and efficient, enabling clients to request precisely the data they need. The server is designed to handle authentication securely, allowing users to create accounts, log in, and engage in private or group chats with confidence. The use of Express.js ensures a smooth and responsive server performance, enhancing the overall user experience.
